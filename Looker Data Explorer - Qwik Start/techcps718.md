### üö® First, click the toggle button to turn on the Development mode.

![Techcps](https://github.com/Techcps/GSP-Short-Trick/assets/104138529/ef540cc4-e6ce-4e81-bf76-75c9ab00a42b)

### üö® Go to Develop > qwiklabs-flights > general > flights > Faa.model

### üö® Faa.model file
- **Add the following code below**

```

explore: +airports {
    query: techcps1 {
      measures: [average_elevation]
    }
  }
  
  explore: +airports {
    query: techcps2 {
      dimensions: [facility_type]
      measures: [average_elevation, count]
    }
  }
    
```
